Class InternalRoutineSlim

This class is a lightweight version of InternalRoutine. Most of exist features part of implementation from InternalRoutineSlim got revised and optimized to avoid extra cost for performance.

InternalRoutineSlim runs PSTk.Threading.Tasks.InternalRoutineSlim.TickCore process on a attached on main to avoid thread-unsafe issues.
